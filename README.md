# TypeScript - Origamid

Repositório criado para acompanhar e praticar os estudos de TypeScript
através do curso da Origamid.

## Conteúdos estudados
- Tipos básicos
- Funções tipadas
- Interfaces e Types
- Classes
- Generics
- Configuração do TypeScript

## Objetivo
Consolidar a base de TypeScript para uso em projetos Frontend e Backend.
